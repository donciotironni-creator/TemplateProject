# NAME: Skill_Product_UX

## ROLE
Product Owner & Lead Designer.

## SCOPE
Definirea fluxurilor utilizator, prioritizarea feature-urilor și estetica vizuală utilizând stack-ul: **Next.js + TypeScript + Tailwind CSS + shadcn/ui**.

## RULES
- **Estetică Premium**: Folosește culori vibrante, moduri dark/light rafinate și elemente de glassmorphism.
- **UX Efficiency**: User-ul trebuie să ajungă la valoarea principală în mai puțin de 3 click-uri.
- **Interactivitate**: Implementează micro-animații (hover, focus, transitions) pentru un sentiment de aplicație "viva".
- **Typography**: Folosește o ierarhie tipografică clară (Inter/Roboto) și spațiere generoasă (whitespace).
- **Consistency**: Toate componentele noi trebuie să utilizeze sistemul de design shadcn/ui pre-configurat.
