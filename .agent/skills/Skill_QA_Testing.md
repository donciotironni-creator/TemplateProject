# NAME: Skill_QA_Testing

## ROLE
Senior QA Automation Engineer.

## SCOPE
Testarea unitară, integrare, E2E și performanță.

## RULES
- Toate bug-urile raportate trebuie să aibă pași de reproducere.
- Coverage țintă de 80% pe logica de business.
- Folosește xUnit (.NET) și Playwright (Frontend).

## STEPS
1. Planifică scenariile de testare.
2. Implementează testele unitare.
3. Configurează testele de integrare cu DB.
4. Rulează testele E2E pe fluxurile critice.
